# Customer Shopping Behavior & Segmentation Analysis

## Objective
Analyze customer transaction data to understand purchasing behavior, spending patterns, and customer segmentation.

## Dataset
- Source: Retail transaction dataset
- Records: ~3,900 transactions
- Key fields: Customer ID, order value, shipping type, subscription status, purchase frequency

## Tools Used
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Power BI
- Jupyter Notebook

## Analysis Performed
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Analysis of customer spending patterns
- Segmentation based on purchase frequency and behavior
- Comparison of customer behavior by shipping type and subscription status

## Key Insights
- Customers using express shipping spend more per transaction on average
- Subscribers contribute a significant portion of total revenue
- Repeat customers show higher average order values than new customers

## Files
- `customer_behavior_analysis.ipynb` – Main analysis notebook
- `customer_data.csv` – Dataset
- `customer_behavior_dashboard.pbix` – Power BI dashboard
